# Minecraft Pets Clicker

Роль в проекте: Front-End Unity Developer

Сборка под Андройд, PC

Милая игра-кликер эволюция с 3D-графикой, как в «Майнкрафте». 
Кликай по питомцу, накапливай монетки, прокачивай клики.
